Power Grid Graph Prototype

A simple website presenting the graph representation and mathematical model of an electrical power grid.

Graph Representation and Prototype Design

The electrical power grid is represented as an undirected graph where:

  • Vertices represent power stations and substations
  • Edges represent transmission lines connecting them
  • An undirected graph is used because electricity can flow in both directions

Mathematical Model of the Power Grid

The power grid is mathematically represented as a graph:

G = (V, E)

Where:

  • G = Graph representing power grid
  • V = Set of vertices (power stations and substations)
  • E = Set of edges (transmission lines)
Vertex Set: 
V = {P1, P2, S1, S2, S3}
Edge Set: 
E = {(P1,S1), (P2,S1), (S1,S2), (S2,S3)}

Degree of Each Vertex

  • deg(P1) = 1
  • deg(P2) = 1
  • deg(S1) = 3
  • deg(S2) = 2
  • deg(S3) = 1

Node S1 has the highest degree → most important node.

Conclusion

  • Graph helps analyze power grid structure
  • Helps identify critical stations and connections
  • Used to study reliability and vulnerability of the network
